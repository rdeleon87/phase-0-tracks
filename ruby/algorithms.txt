#how to change guitar strings on a hard tail bridge

# Assemble tools you will need (string winder, phillips head screwdriver, and pliers)
# Loosen tension on all strings
# Begin unwinding strings one at a time with the winder
# once strings a removed, clean the fretboard and bridge
# Begin winding with the lowest string and go up from there
# tune each string
# Stretch strings by playing and bending them for a few minutes 
# tune again and repeat the process until tunin stabililty is achieved
# Throw out old strings
